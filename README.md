# PediCalc
PediCalc: Calculadora médica pediátrica diseñada para profesionales de la salud. Calcula dosis, líquidos, peso ideal, tubos endotraqueales, escalas clínicas y parámetros anestésicos de forma rápida, precisa e intuitiva. Herramienta práctica para la toma de decisiones en urgencias, hospitalización y quirófano.
